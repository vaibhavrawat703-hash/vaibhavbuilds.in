Warehouse Management System Redesign

Owned the redesign of a WMS around 7 real user personas instead of a generic feature list — restructuring inbound, putaway, picking, packing, and billing around actual operator workflows. Made the case for AI-agent-assisted task execution in picking/packing, prioritizing it over manual SOP updates because it addressed the root workflow gap, not just a symptom.

Real-Time Ops Dashboard for Account Managers

Owned the product definition for a dashboard replacing a workflow that ran entirely through phone calls to ops teams. Deliberately scoped it as visibility-only, no in-system actions — account managers needed to see status, not take action inside the tool — a scope call that kept the build lean and matched how the team actually worked.
